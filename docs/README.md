# ESA HttpClient

ESA HttpClient is an asynchronous event-driven http client based on netty which supports

## Features

- Http1/H2/H2cUpgrade
- Https
- Epoll/NIO
- Interceptor
- Filter
- Retry, Redirect, 100-expect-continue
- Chunked read/write
- Multipart
- Metrics
- more ...
