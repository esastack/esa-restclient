# ESA HttpClient

ESA HttpClient is an asynchronous event-driven http client based on netty which supports.

## Features

- Http1/H2/H2cUpgrade
- Https
- Epoll/NIO
- Interceptor
- Serialize and Deserialize([RestClient](./Reference Doc/v1.0.0/RestClient)
- Filter
- Retry, Redirect, 100-expect-continue
- Multipart
- Metrics
- more ...
