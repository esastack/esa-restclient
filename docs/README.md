# ESA RestClient

ESA RestClient is an asynchronous event-driven http client based on netty which supports.

## Features

- Http1/H2/H2cUpgrade
- Https
- Epoll/NIO
- Serialize and Deserialize
- Interceptor
- Filter
- Retry, Redirect, 100-expect-continue
- Multipart
- Metrics
- more ...
